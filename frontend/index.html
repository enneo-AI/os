Chat schließen

Bei „Lernen & Schließen" zieht Enni dauerhafte Learnings aus dieser Konversation — sie wirken sofort für dich, der Admin entscheidet über die Team-weite Übernahme.

Attio verbinden

Attio → Workspace Settings → Developers → Access Tokens. Ein Key mit Lese-Rechten reicht — Enni liest nur (Accounts, Kontakte, Deals, Notizen), schreibt nie. Der Key ist nach dem Speichern aus der App nicht mehr auslesbar (write-only, nur das Backend nutzt ihn).

Seite von URL importieren

Die Seite wird gecrawlt, als Markdown gespeichert und sofort für Ennis Suche indexiert. Der Import dauert ein paar Sekunden.

Slack verbinden

Einmalig unter api.slack.com/apps eine App im Workspace anlegen → OAuth & Permissions → Bot Token Scopes: channels:read, channels:history, channels:join, groups:read, groups:history, users:read → Install to Workspace → den xoxb-Token hier einfügen. Enni liest nur — öffentliche Channels automatisch, private nur nach Einladung des Bots. Der Token ist nach dem Speichern aus der App nicht mehr auslesbar.
